## Usage

```tsx
import { Slider } from "@/components/ui/slider"
```

```tsx
<Slider defaultValue={[33]} max={100} step={1} />
```

### Range Slider

You can use the `SliderThumb` component to enable selection of multiple values within a range using additional thumbs.

```tsx
import { Slider, SliderThumb } from "@/components/ui/slider"
```

```tsx
<Slider defaultValue={[20, 80]} max={100} step={1}>
  <SliderThumb />
  <SliderThumb />
</Slider>
```
